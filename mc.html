About:

SeanCraft is Hermitcraft like whitelist server hosted by me,

we are looking for people who are either good with redstone or builders in order to create a good looking world the is also functional

The server is constantly changing, current plugins/datapacks include: Chestshop, Vault, Essentials, PermissionsEX

One player sleep, anti ender grief, and more

The main goal of this server is to automate as much of the game as possible and to create a fun place to interact .

The server will be available for people to join in a few months, A discord server will be open in a few weeks.

More information such as rules and guides will be updated soon.

This is a dedicated server that will be upgraded soon, currently the server has:

Dual core CPU

4gb dedicated ram

120GB ssd

Gigabyte ethernet link

Ubuntu Gnome
